# College-Management-System

A web-based College Management System developed to automate and manage academic and administrative operations efficiently.

## Features

- Student Record Management
- Faculty Management
- Course Management
- Attendance Tracking
- Result Management
- Admin Dashboard
- Secure Login and Logout
- Database Integration

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Modules

### Admin Module
- Manage Students
- Manage Faculty
- Manage Courses
- Track Attendance
- Publish Results

### User Module
- View Student Details
- Access Academic Records
- Login Authentication

## Project Setup

1. Clone Repository

```bash
git clone https://github.com/pratibhaksingh98/College-Management-System.git
```

2. Import database file (`bit.sql`) into MySQL

3. Run project on localhost using XAMPP/WAMP
